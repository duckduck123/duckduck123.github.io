Hexo+github搭建个人博客

Hexo 是一款基于Node.js、快速、简洁且高效的博客框架。Hexo 使用Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页，Nodejs框架是基于V8的引擎，是目前速度最快的Javascript引擎，GitHub Pages本用于介绍托管在GitHub的项目， 不过，由于他的空间免费稳定，可用来做搭建一个博客。

接下来进入教程

1、环境配置

node.js 环境配置 https://nodejs.org/en/download/
git 环境配置 https://git-scm.com/
软件安装一直下一步即可

2、安装hexo

C:\Users\XYuser>d:
D:>cd D:\Program Files\nodejs //进入nodejs安装目录
D:\Program Files\nodejs>npm install -g hexo
稍等几分钟就安装好了……
D:\Program Files\nodejs>mkdir myblog //在nodejs安装目录创建你的博客名称
D:\Program Files\nodejs>cd myblog
D:\Program Files\nodejs\myblog>hexo init //初始化
D:\Program Files\nodejs\myblog>npm install //安装依赖包
D:\Program Files\nodejs\myblog>hexo g //生成静态页面
D:\Program Files\nodejs\myblog>hexo s //启动
你的博客已经搭建完成访问http://localhost:4000即可看到你的博客

接下来在github上创建自己的主页
